This is a plugin for updating the label of title field for each post type
== Description ==
This plugin allows you to change and update the title labels for each post type.

After activating the plugin, you can access it (‘SIC Change Title’) from the settings menu. Here you will find a list of all post types in the system alongside a text box where you can enter your custom label. If you choose to leave it as blank, the default label from WordPress (Title) will be used.
== Installation ==
1. Upload ”SIC Change Title Label” to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
